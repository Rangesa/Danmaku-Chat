# AI臭い表現リスト

記事執筆時に**絶対に使用してはいけない表現**のリスト。

## 過剰な丁寧語

### 禁止表現
- 「〜してみてください」
- 「〜してみましょう」
- 「〜してみてはいかがでしょうか」
- 「ぜひ〜してください」
- 「ぜひお試しください」

### 代替表現
- 「〜すると良い」
- 「〜した」
- 「〜するのがおすすめ」

## 機械的な接続詞

### 禁止表現
- 「まず、〜」「次に、〜」「最後に、〜」の3点セット
- 「以下の通りです」
- 「下記をご確認ください」
- 「それでは〜していきます」
- 「〜について解説していきます」

### 代替表現
- 自然な文脈で繋ぐ
- 番号付きリストは本当に必要な時のみ

## 不自然な強調

### 禁止表現
- 絵文字の使用全般（✨🎉💡👍など）
- 過剰な太字
- 「！」「！！」の乱用
- 中黒「・」による不必要なリスト化

### 代替表現
- 顔文字（適切な場所のみ）
- 自然な文の強弱

## 浅い締め方

### 禁止表現
- 「いかがでしたでしょうか」
- 「参考になれば幸いです」
- 「ぜひ活用してみてください」
- 「〜してみてはいかがでしょうか」

### 代替表現
- 「良かったら使ってみてください」
- 「GitHubに置いてあります」
- 自然に終わる（締めの挨拶は不要な場合も多い）

## 曖昧な表現

### 禁止表現
- 「〜かもしれません」の連発
- 「〜と思われます」
- 「〜の可能性があります」（断言できる内容で）

### 代替表現
- 自信がある部分は断言
- 本当に不確かな部分だけ留保

## 機械的なリスト

### 禁止パターン
```
メリット：
・〜
・〜
・〜

デメリット：
・〜
・〜
```

### 代替パターン
```
便利だった点は○○と△△。特に○○は予想以上だった。

ただ、まだ××の部分が改善の余地ありそう。
```

## 説教臭い言い回し

### 禁止表現
- 「重要なのは〜」
- 「ポイントは〜」
- 「注意すべきは〜」
- 「覚えておきましょう」

### 代替表現
- 「〜で躓いた」
- 「〜が難しかった」
- 経験を語る形式

## チェックリスト

執筆後、以下を確認：

- [ ] 「〜してみてください」を使っていないか
- [ ] 「まず」「次に」「最後に」の3点セットになっていないか
- [ ] 絵文字を使っていないか
- [ ] 不必要な箇条書きになっていないか
- [ ] 「いかがでしたでしょうか」で締めていないか
- [ ] 具体的なエピソードが含まれているか
- [ ] 失敗談や試行錯誤が書かれているか
