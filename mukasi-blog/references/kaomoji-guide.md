# 顔文字使用ガイド

記事内で使用できる顔文字と、その適切な使用方法。

## 基本原則

1. **控えめに使う**：1段落に1個以下、見出しごとに0〜1個程度
2. **自然な場所に配置**：感情が動いた場所、リアクションを示す場所
3. **絵文字は使わない**：あくまでテキストベースの顔文字のみ

## 顔文字カテゴリー

### 困った・悩んだ系
- `(´・ω・`)` - ショボーン、困った時
- `(´；ω；`)` - 泣いてる、本気で困った
- `(´-ω-`)` - うーん...と考え込む

**使用例**
```
既存のMODだと設定画面が使いづらくて(´・ω・`)
```

### 満足・達成系
- `('ω')` - もきゅもきゅ、満足
- `(｀･ω･´)` - シャキーン、やる気
- `( ˘ω˘ )` - スヤァ、安心

**使用例**
```
結局自分で作ることにした('ω')
よし、実装するぞ(｀･ω･´)
```

### リアクション系
- `|дﾟ)` - チラ見、様子見
- `(ﾉ∀`)` - バレテーラ、ニヤニヤ
- `(ﾟ∀ﾟ)` - ワクワク、興奮

**使用例**
```
予想以上に便利で自分でも驚いた|дﾟ)
```

### テンション高い系
- `ｷﾀ━━━━(ﾟ∀ﾟ)━━━━!!` - キター！
- `ｷﾀ━(ﾟ∀ﾟ)━!` - 短縮版
- `ﾊﾞﾚﾃｰﾗ(ﾉ∀`)` - バレてーら

**使用例**
```
ビルド成功！ｷﾀ━(ﾟ∀ﾟ)━!
```
（注：テンション高い系は使いすぎ注意。本当に嬉しい時だけ）

### その他よく使われるもの
- `(´ー`)` - フーン、まぁいいか
- `(・∀・)` - イイ！
- `( ´_ゝ`)` - フーン
- `_(:3 」∠)_` - 寝落ち
- `(　ﾟдﾟ)` - ポカーン

## 使用頻度の目安

### 良い使用例（適切な頻度）

```
タイトル：Minecraft用の便利ツール作った

最近○○で困ってたんですよね(´・ω・`)

[段落×3]

で、作ってみた('ω')

[段落×5]

意外と上手くいって驚いた|дﾟ)

[段落×3、顔文字なし]

GitHubに上げたので良かったら使ってください
```

### 悪い使用例（使いすぎ）

```
こんにちは！('ω')
今日は〜を紹介します(｀･ω･´)
まず〜(´・ω・`)
次に〜('ω')
最後に〜|дﾟ)
```

## 配置のコツ

### ○ 良い配置
- 感情が動いた瞬間の文末
- 予想外の出来事の後
- 失敗や困った時の描写

### × 悪い配置
- 見出しの直後
- 連続する文章の各文末
- 技術的な説明の途中

## 使わない方が良い場所

- 技術詳細の説明中
- コードブロックの前後
- 真面目なトラブル報告
- GitHubリンクなどの案内部分

## 禁止事項

以下の顔文字風のものは使用禁止：
- 西洋式の顔文字： `:)` `:(` `:D` など
- 絵文字全般： 😀 😊 🎉 など
- 古すぎる2ch語： `( ´∀`)ﾓﾅｰ` など（文脈次第では可）
